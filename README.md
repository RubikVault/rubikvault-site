# RubikVault Landing Page

Static landing page for **rubikvault.com**, deployed via Cloudflare Pages.

- Dark themed, responsive layout
- Embedded TradingView widgets (ticker tape, mini chart, crypto overview, economic calendar)
- Legal pages: Imprint, Privacy, Disclaimer
- No build step required (pure HTML + CSS)

## Structure

- `index.html` – main landing page
- `style.css` – global styling
- `imprint.html` – German legal imprint
- `privacy.html` – privacy policy (simplified template)
- `disclaimer.html` – no-investment-advice disclaimer

## Local development

Just open `index.html` in a browser.  
No bundler, no framework, no external build tooling required.
