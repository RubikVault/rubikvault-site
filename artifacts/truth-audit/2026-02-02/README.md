# Truth Audit Artifacts (2026-02-02)

This folder contains evidence artifacts generated for the "Truth Path Audit" runbook.
All files are read-only evidence; no production logic depends on them.

Generated by:
- scripts/truth-audit/fetch_raw.mjs
- scripts/truth-audit/static_scan.mjs
- scripts/truth-audit/run_scenarios.mjs

Key files:

- STATIC_REFERENCES.json: static token reference map
- RUNTIME_TELEMETRY.json: runtime path hits from scenarios
- SCENARIO_MATRIX.md: scenario outcomes
- INVENTORY.md: human-readable contract + path inventory
- DELETION_PLAN.md: deletion candidates (no deletions performed)
